Страница¶

Использование¶

Начинайте каждую сцену с ключевого слова Страница с названием.

Page "Signup"
Block "Top menu"
    ...
Block "Personal data"
    ...
Block "Footer"

Название страницы поможет вам запомнить, что описывает эта сцена. Оно не выводится на мокап.

Порядок слов и заглавные буквы опциональны. Используйте их, как вам удобно:

"Signup" page
...
